# Sangwa Jesly - Portfolio

A modern, editorial-style portfolio website showcasing the work of Sangwa Jesly, a Software Engineer and Marketing Designer specializing in web development, brand design, and AI automations.

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom design system
- **Build Tool**: Vite
- **Routing**: React Router
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Features

- Responsive design with dark theme
- Modular component architecture
- Case study format for projects
- Blog and resources sections ready for expansion
- Clean typography with custom fonts (Inter, Clash Display, Newsreader, Space Mono)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sangwajesly/sangwa-jesly.git
   cd sangwa-jesly
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

```

```

## Deployment

The site is built with Vite and can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.
